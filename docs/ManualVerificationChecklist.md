# NightScope 手動検証チェックリスト

最終更新: 2026-04-02
対象: macOS 26+

## 1. UI状態遷移

- [ ] 初回起動時、`DetailView` が「計算中プレースホルダー」→「結果表示」へ遷移する
- [ ] 計算結果なし時に `ContentUnavailableView` が表示される
- [ ] 位置を変更すると、ヘッダー・観測ウィンドウ・2週間グリッドが再計算される
- [ ] 日付変更で表示内容が即座に切り替わる

## 2. サイドバー操作

- [ ] 検索欄で入力すると候補が表示される
- [ ] ↑/↓キーで候補ハイライト移動、Enterで確定できる
- [ ] Escキーで検索入力とハイライトがクリアされる
- [ ] 「地図 / 光害」セグメント切り替えで表示が崩れない
- [ ] 現在地ボタン押下でピン・表示範囲が更新される

## 3. 地図タイル（重点）

- [ ] 光害表示ON時、タイルが表示される
- [ ] パン/ズーム中に白抜け・激しいちらつきが発生しない
- [ ] タイルの再描画遅延が体感上悪化していない（旧版比較）
- [ ] 光害表示OFF時、オーバーレイが非表示になる
- [ ] タブ往復後もタイル状態が維持される

## 4. エラー導線

- [ ] 光害取得失敗時にエラーバナーが表示される
- [ ] 天気取得失敗時にエラーバナーが表示される
- [ ] 「再試行」ボタンで復帰できる

## 5. アクセシビリティ

- [ ] VoiceOverで主要UI（検索、現在地、日付、カード）が読み上げ可能
- [ ] アイコンのみボタンに適切な `accessibilityLabel` がある
- [ ] 「透明度を下げる」ONでも可読性が維持される
- [ ] 「視差効果を減らす」ONで不自然なアニメーションにならない

## 6. キーボード操作

- [ ] ⌘R で更新（設定済みショートカット）が動作する
- [ ] ⌘F で検索フォーカス移動（設定済みの場合）
- [ ] Tab移動順が論理的（検索→候補→地図→日付）

## 7. 回帰確認（機能）

- [ ] 星空指数（0-100）が表示される
- [ ] 月相表示（名称・アイコン）が正しい
- [ ] 観測可能ウィンドウ表示（天候/月明かり考慮）が正しい
- [ ] 2週間グリッドに天気・月・指数が表示される
